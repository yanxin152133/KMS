## 1.2. Office激活
### 1.2.1. 注意事项
- 首先你的Office必须是VOL版本，否则无法激活。
- Office安装目录，比如C:\Program Files (x86)\Microsoft Office\Office16
- 64位的就是C:\Program Files\Microsoft Office\Office16
- office16是对应的office2016版本，office15是对应的Office2013版本，office14是对应的Office2010。
